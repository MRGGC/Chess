## Tools and Languages used

* Node JS (server)
* Electron JS (closs-platform desktop app)
* Web Sockets (multiplayer)
* ES6 (game mechanics)
* p5.js (graphics)
* HTML/CSS, JavaScript, jQuery, w3.css (UI/UX)
